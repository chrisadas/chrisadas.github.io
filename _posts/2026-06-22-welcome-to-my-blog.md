This is my first post. The site runs on [Jekyll](https://jekyllrb.com/)
with the [Minima](https://github.com/jekyll/minima) theme, hosted free on
GitHub Pages.

## How to write a new post

1. In the repo, open the `_posts/` folder on github.com.
2. Click **Add file → Create new file**.
3. Name it `YYYY-MM-DD-some-title.md` — the date and `.md` matter.
4. Paste this header at the top (called *front matter*):

   ```yaml
   ---
   layout: post
   title: "Your title here"
   date: 2026-06-22 10:00:00 +0000
   categories: general
   ---
   ```

5. Write the body below it in Markdown.
6. Scroll down, click **Commit changes**.

Within a minute GitHub rebuilds the site and your post appears on the
homepage. No terminal, no build commands.

## Markdown quick reference

- `**bold**` → **bold**, `*italic*` → *italic*
- `## Heading` for section headers
- `[link text](https://example.com)` for links
- Lists with `-`, code with backticks.

Happy writing.
